# my-personal-repo

Personal static files published via **GitHub Pages**.

- **Live site:** https://jjgrane.github.io/my-personal-repo/
- **Published directory:** only `docs/` is served (Pages source = `main` branch, `/docs` folder). The rest of the repo is not published.

## Adding a file

Drop a new `.html` (or asset) into `docs/`, link it from `docs/index.html`, then commit & push:

```sh
git add docs && git commit -m "add <thing>" && git push
```

It goes live at `https://jjgrane.github.io/my-personal-repo/<filename>` within ~1 min.
